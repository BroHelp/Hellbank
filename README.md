# Hellbank
**Project Description**
HellBank is a one of a kind bank for beginners as it has a simple layout that one can understand easily without getting overwhelmed. Although for an online bank the UI is simple and one might even say grotesque but as a bank for begginers we prioritize efficiency than design.
**How to run**
To run the code just simply click BankingApp.java and with the ide that you have be it VS Code, IntelliJ, or Netbeans the code should run fine
